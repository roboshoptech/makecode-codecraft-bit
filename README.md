# makecode-codecraft-bit

Makecode Micro:bit extension for Roboshop Codecraft:bit

## Usage

```blocks
codecraftbit.spinMotor(CCBIT_Motors.CCB_M1, 50);
basic.pause(2000);
codecraftbit.spinMotor(CCBIT_Motors.CCB_M1, -50);
basic.pause(2000);
```
